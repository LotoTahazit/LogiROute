# הסכם רמת שירות — Service Level Agreement (SLA)

## 1. הגדרות

| מונח | הגדרה |
|------|--------|
| שירות | מערכת LogiRoute — ניהול לוגיסטיקה ומסמכים חשבונאיים |
| זמינות | אחוז הזמן שהשירות נגיש ופעיל |
| תקלה | מצב שבו השירות אינו נגיש או אינו מתפקד כמצופה |
| זמן תגובה | הזמן מרגע דיווח על תקלה ועד תחילת טיפול |
| זמן שחזור | הזמן מרגע דיווח ועד חזרה לפעילות מלאה |

## 2. רמת זמינות

| רכיב | SLA | הערות |
|-------|-----|--------|
| אפליקציית Web | 99.9% | ~8.7 שעות downtime בשנה |
| בסיס נתונים (Firestore) | 99.999% | Google SLA |
| אימות (Firebase Auth) | 99.95% | Google SLA |
| API חשבוניות ישראל | best-effort | תלוי ברשות המסים |

## 3. סיווג תקלות

| רמה | תיאור | זמן תגובה | זמן שחזור |
|------|--------|-----------|-----------|
| P1 — קריטי | אין גישה למערכת / אובדן נתונים | 1 שעה | 4 שעות |
| P2 — גבוה | לא ניתן ליצור/להדפיס מסמכים | 4 שעות | 12 שעות |
| P3 — בינוני | תקלה בפיצ'ר לא קריטי | 24 שעות | 72 שעות |
| P4 — נמוך | בקשת שיפור / באג קוסמטי | 72 שעות | לפי תכנון |

## 4. Disaster Recovery

| מדד | ערך |
|------|------|
| RTO (Recovery Time Objective) | < 4 שעות |
| RPO (Recovery Point Objective) | < 1 שעה |
| גיבוי יומי | Firestore Automatic Backups |
| גיבוי רבעוני | Compliance backup + restore test |
| PITR | Point-in-Time Recovery זמין |

## 5. תחזוקה מתוכננת

- תחזוקה מתוכננת: הודעה 48 שעות מראש
- חלון תחזוקה מועדף: ימי שישי 02:00-06:00 (שעון ישראל)
- תחזוקה מתוכננת לא נספרת כ-downtime

## 6. ניטור

| רכיב | כלי |
|-------|------|
| זמינות | Firebase Performance Monitoring |
| שגיאות | Firebase Crashlytics |
| ביצועים | Firestore Usage Dashboard |
| אבטחה | Cloud Audit Logs |

## 7. דיווח

- דוח זמינות חודשי ללקוח
- דוח תקלות רבעוני
- Post-mortem לכל תקלה P1/P2

## 8. פיצוי

| זמינות בפועל | פיצוי |
|---------------|--------|
| < 99.9% | 10% הנחה בחודש הבא |
| < 99.5% | 25% הנחה בחודש הבא |
| < 99.0% | 50% הנחה בחודש הבא |
| < 95.0% | חודש חינם |

## 9. חריגים

SLA לא חל על:
- תקלות ברשות המסים (API חשבוניות ישראל)
- תקלות ב-Google Cloud infrastructure
- שימוש לא מורשה או חריג
- Force majeure
