# מבנה בסיס נתונים

> **מסמך זה נדרש להגשה לרישום תוכנה לניהול ספרים (מס הכנסה)**

## טכנולוגיה

- **מסד נתונים**: Google Cloud Firestore (NoSQL, serverless)
- **אזור**: `nam5 (United States)` — Google Cloud multi-region
- **אימות**: Firebase Authentication
- **אחסון**: Firebase Storage (לוגואים, קבצים)
- **אירוח**: Firebase Hosting (Web), APK (Android)

## מבנה אוספים

### אוספים ראשיים
```
users/{userId}                          — משתמשים
companies/{companyId}                   — חברות
driver_locations/{driverId}             — מיקומי נהגים (GPS)
```

### תת-אוספים של חברה
```
companies/{companyId}/
  ├── settings/{settingId}              — הגדרות חברה
  ├── members/{uid}                     — חברי צוות
  ├── invites/{inviteId}                — הזמנות
  │
  ├── accountingDocs/{docId}            — מסמכים חשבונאיים (חדש)
  │
  ├── accounting/_root/
  │   ├── counters/{docType}            — מוני מספור אטומי (חדש)
  │   ├── invoices/{invoiceId}          — חשבוניות (legacy)
  │   ├── receipts/{receiptId}          — קבלות (legacy)
  │   ├── credit_notes/{creditId}       — זיכויים (legacy)
  │   ├── integrity_chain/{chainId}     — שרשרת שלמות (append-only)
  │   ├── integrity_anchors/{anchorId}  — עוגני שלמות (append-only)
  │   └── assignment_requests/{reqId}  — בקשות מספר הקצאה
  │
  ├── audit/{eventId}                   — יומן ביקורת cross-module (append-only)
  ├── backups/{backupId}                — רישומי גיבוי (append-only)
  ├── restore_tests/{testId}            — בדיקות שחזור (append-only)
  │
  ├── warehouse/_root/
  │   ├── box_types/{boxTypeId}
  │   ├── product_types/{productTypeId}
  │   ├── inventory/{itemId}
  │   ├── inventory_counts/{countId}
  │   └── inventory_history/{historyId}
  │
  └── logistics/_root/
      ├── clients/{clientId}
      ├── delivery_points/{pointId}
      ├── cached_routes/{routeId}
      └── prices/{priceId}
```

## מסמך חשבונאי (AccountingDoc)

**אוסף**: `companies/{companyId}/accountingDocs/{docId}`

### שדות ליבה
| שדה | סוג | תיאור |
|------|------|--------|
| `type` | String | סוג: `tax_invoice`, `receipt`, `tax_invoice_receipt`, `credit_note` |
| `status` | String | סטטוס: `draft`, `issued`, `locked`, `credited`, `voided_before_delivery` |
| `docNumber` | int? | מספר רץ (מוקצה בהנפקה) |
| `issuedAt` | Timestamp? | חותמת זמן הנפקה (serverTimestamp) |
| `customerId` | String | מזהה לקוח |
| `customerName` | String | שם לקוח |
| `customerTaxId` | String? | ח.פ. / ת.ז. לקוח |
| `lines` | Array | שורות המסמך |
| `totals` | Map | סיכומים: `net`, `vat`, `gross` |
| `references` | Map? | קישורים: `originalDocId`, `originalDocNumber`, `creditNoteIds` |
| `reason` | String? | סיבת זיכוי (חובה ב-credit_note) |
| `createdAt` | Timestamp | תאריך יצירה (serverTimestamp) |
| `createdBy` | String | UID יוצר |
| `companyId` | String | מזהה חברה |
| `immutableSnapshotHash` | String? | SHA-256 (מוקצה בהנפקה) |
| `updatedAt` | Timestamp? | תאריך עדכון אחרון |
| `updatedBy` | String? | UID מעדכן |
| `notes` | String? | הערות |

### שדות שורה (AccountingDocLine)
| שדה | סוג | תיאור |
|------|------|--------|
| `description` | String | תיאור פריט |
| `quantity` | double | כמות |
| `unitPrice` | double | מחיר ליחידה |
| `totalBeforeVat` | double | סכום לפני מע"מ |
| `vatRate` | double | שיעור מע"מ (ברירת מחדל: 0.17) |
| `vatAmount` | double | סכום מע"מ |
| `totalWithVat` | double | סכום כולל מע"מ |

### שדות סיכום (AccountingDocTotals)
| שדה | סוג | תיאור |
|------|------|--------|
| `net` | double | סכום נטו |
| `vat` | double | סכום מע"מ |
| `gross` | double | סכום ברוטו |

## מונה מספור (Counter)

**נתיב**: `companies/{companyId}/accounting/_root/counters/{docType}`

| שדה | סוג | תיאור |
|------|------|--------|
| `lastNumber` | int | המספר האחרון שהוקצה |

## הפרדת דיירים (Tenant Isolation)

- כל הנתונים מאוחסנים תחת `companies/{companyId}/`
- Firestore Security Rules מוודאים:
  - `userDoc().companyId == companyId` — גישה רק לנתוני החברה שלך
  - `super_admin` — גישה לכל החברות
- אין שאילתות חוצות-חברות
- `companyId` נלקח מ-`CompanyContext.effectiveCompanyId` (לא hardcoded)
