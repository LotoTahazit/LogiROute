|------|------|

## תיאור כללי

LogiRoute היא מערכת SaaS לניהול לוגיסטיקה, הפצה ותיעוד חשבונאי.
המערכת מיועדת לחברות הפצה ושינוע בישראל ומספקת:

- ניהול מסלולי הפצה ונקודות מסירה
- ניהול מלאי ומחסן
- הפקת חשבוניות מס, קבלות, תעודות משלוח וזיכויים
- מעקב GPS אחר נהגים בזמן אמת
- ניהול לקוחות ומחירונים
- דוחות וסיכומים יומיים

## מבנה ארגוני

המערכת תומכת בהפרדה מלאה בין חברות (Tenant Isolation):
- כל חברה מזוהה ב-`companyId` ייחודי
- כל הנתונים מאוחסנים בתת-אוספים: `companies/{companyId}/...`
- אין גישה חוצת-חברות (Cross-Tenant)

## תפקידים במערכת

| תפקיד | הרשאות |
|--------|---------|
| super_admin | גישה מלאה לכל החברות |
| admin | ניהול חברה ספציפית |
| dispatcher | הפקת חשבוניות, ניהול מסלולים |
| driver | צפייה במסלול, עדכון סטטוס משלוח |
| warehouse_keeper | ניהול מלאי |
