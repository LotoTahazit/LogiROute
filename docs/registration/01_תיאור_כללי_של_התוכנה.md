## תיאור כללי

LogiRoute היא מערכת SaaS לניהול מסמכים חשבונאיים, לוגיסטיקה והפצה.
המערכת מיועדת לחברות הפצה ושינוע בישראל.

**המודול הרלוונטי לרישום תוכנה לניהול ספרים הוא מודול החשבונאות בלבד.**

### מודול החשבונאות — תיאור

מודול החשבונאות מאפשר:
- הפקת מסמכים חשבונאיים: חשבונית מס, קבלה, חשבונית מס קבלה, תעודת זיכוי
- מספור רץ אטומי ורציף לכל סוג מסמך
- הקפאת מסמכים (draft → issued) עם חתימת שלמות SHA-256
- מניעת שינוי ומחיקה לאחר הנפקה
- יומן ביקורת append-only לכל פעולה
- הפקת קובץ במבנה אחיד (CSV) לרשות המסים
- תמיכה במספר הקצאה (חשבוניות ישראל)

## מבנה ארגוני

המערכת תומכת בהפרדה מלאה בין חברות (Tenant Isolation):
- כל חברה מזוהה ב-`companyId` ייחודי
- כל הנתונים מאוחסנים בתת-אוספים: `companies/{companyId}/...`
- אין גישה חוצת-חברות (Cross-Tenant)
- Firestore Security Rules אוכפים הפרדה ברמת שרת

## תפקידים במערכת

| תפקיד | קוד | הרשאות רלוונטיות לחשבונאות |
|--------|------|---------------------------|
| סופר-אדמין | `super_admin` | גישה מלאה לכל החברות |
| בעלים | `owner` | קריאה בלבד לדוחות ומסמכים |
| מנהל | `admin` | ניהול מלא של מסמכים חשבונאיים |
| רואה חשבון | `accountant` | יצירה ועדכון מסמכים חשבונאיים (draft בלבד), קריאת דוחות ויומן ביקורת |
| מוקדן | `dispatcher` | הפקת מסמכים, ניהול מסלולים |
| נהג | `driver` | ללא גישה למסמכים חשבונאיים |
| מחסנאי | `warehouse_keeper` | ללא גישה למסמכים חשבונאיים |

> **תפקיד `accountant`** — תפקיד ייעודי לרואה חשבון: גישה מוגבלת למודול החשבונאות בלבד, ללא גישה ללוגיסטיקה, מחסן או ניהול חברה.

## ארכיטקטורה טכנית

### שכבות הגנה

1. **Firebase Authentication** — אימות JWT בכל בקשה
2. **Firestore Security Rules** — אכיפת הרשאות ברמת שרת (לא ניתן לעקוף)
3. **Application Layer** — בדיקות תקינות בצד לקוח (UX בלבד)
4. **Firestore Transactions** — אטומיות מספור ועדכון מסמכים

### אחסון נתונים

- **מסד נתונים**: Google Cloud Firestore (NoSQL, serverless)
- **אזור**: nam5 (United States) — Google Cloud multi-region
- **הצפנה**: AES-256 at-rest, TLS 1.2+ in-transit (Google-managed)
- **גיבוי**: PITR (Point-in-Time Recovery, 7 ימים) + גיבוי יומי (14 ימים) + גיבוי רבעוני לתאימות
