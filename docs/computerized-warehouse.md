# מחסן ממוחשב — кратко для команды

Включение: **Owner Dashboard → הגדרות → הגדרות מערכת → «מחסן ממוחשב (ברקודים)»** (только admin) → сохранить. После этого в меню **מחסן** появляется пункт **סריקת ברקוד**; на карточках товаров — кнопка редактирования штрихкода. Сканирование ищет позицию по **מק"ט** (код товара / id документа) или по полю **barcode** (EAN), заданному при добавлении или через «עדכון ברקוד»; один штрихкод — одна позиция. Каждая операция прихода/расхода через скан пишется в `companies/{companyId}/warehouse/_root/inventory_history`: `action` = `barcode_in` или `barcode_out`, `reason` = `barcode_scan`, поле `barcode` = отсканированный код, плюс стандартные поля (`productCode`, `quantityChange`, `quantityBefore/After`, `userName`, `timestamp`). В **דוח שינויים במלאי** такие записи отображаются с иконкой QR.
