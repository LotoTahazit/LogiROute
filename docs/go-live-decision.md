## Решение

| 🟢 (из 22) | 🔴 (из 22) | Решение |
|------------|------------|---------|
| | | |

- **22/22 🟢** → Go-Live: первый платящий клиент + микро-пилот  
- **≥ 19 🟢, 🔴 только E20 (+ незаполненные A/B без процесса)** → **микро-пилот с white-glove**, drill до или в неделю 1  
- **E20 🔴 без плана drill** → **No-Go** на платящего клиента  
- **≥ 2 🔴 в блоке A** → **No-Go** — нет процесса внедрения
