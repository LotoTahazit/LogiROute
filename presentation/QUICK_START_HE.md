## שלב 3: צלם צילומי מסך (1-2 שעות)

### פתח את הקובץ: `06_SCREENSHOTS_CHECKLIST.md`

### צור תיקיות לצילומי מסך:
```bash
mkdir presentation/screenshots
mkdir presentation/screenshots/login
mkdir presentation/screenshots/admin
mkdir presentation/screenshots/dispatcher
mkdir presentation/screenshots/driver
mkdir presentation/screenshots/warehouse
mkdir presentation/screenshots/misc
mkdir presentation/screenshots/diagrams
```

### צילומי מסך בעדיפות גבוהה (מינימום 30):

#### חובה:
1. **Login** (3 צילומי מסך)
   - מסך כניסה
   - בחירת שפה
   - טופס מלא

2. **Admin** (5 צילומי מסך)
   - מסך ראשי
   - יצירת משתמש
   - עריכת נהג
   - מצב "View As"
   - רשימת משתמשים

3. **Dispatcher** (12 צילומי מסך)
   - שלוש הכרטיסיות
   - יצירת נקודה
   - רשימת נקודות
   - מסלולים לפי נהגים
   - מפה עם מסלולים
   - סמנים במפה
   - סינון לפי נהגים
   - Drag & Drop

4. **Driver** (6 צילומי מסך)
   - מסך ראשי
   - כפתור ניווט
   - GPS פעיל
   - נקודה עם כפתור "הושלם"
   - מפה עם מסלול
   - נקודות שהושלמו

5. **Warehouse** (4 צילומי מסך)
   - מסך ראשי
   - רשימת מוצרים
   - הוספת מוצר
   - דוח

### איך לצלם:

#### Android:
```
1. פתח את המסך הנדרש
2. לחץ Power + Volume Down
3. צילום המסך יישמר בגלריה
4. העבר ל-presentation/screenshots/
```

#### Web:
```
1. פתח DevTools (F12)
2. הפעל Device Toolbar (Ctrl+Shift+M)
3. בחר רזולוציה 1920x1080
4. צלם (Ctrl+Shift+P → "Capture screenshot")
5. שמור ב-presentation/screenshots/
```

### חשוב: בחר שפה אחת!
⚠️ כל צילומי המסך חייבים להיות באותה שפה:
- עברית (מומלץ ל-Y.C. Plast)
- רוסית (אם הקהל דובר רוסית)
- אנגלית (לקהל בינלאומי)

אל תערבב שפות במצגת אחת!

### דרישות לצילומי מסך:
- ✅ רזולוציה: מינימום 1920x1080
- ✅ פורמט: PNG
- ✅ **שפה: אחת לכל המצגת** (עברית או רוסית או אנגלית)
- ✅ תמונה ברורה
- ❌ ללא מידע אישי
- ❌ ללא שגיאות במסך
- ❌ אל תערבב שפות!
