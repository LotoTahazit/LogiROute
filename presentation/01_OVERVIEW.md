# 🚚 LogiRoute - Обзор системы

## Что это?

**LogiRoute** - это комплексная система управления логистикой и грузоперевозками, разработанная для компании Y.C. Plast.

## Основная информация

- **Название**: LogiRoute
- **Версия**: 1.0.0 (стабильная)
- **Компания**: Y.C. Plast
- **Платформы**: Android, Web
- **Технологии**: Flutter, Firebase

## Ключевые возможности

### 📱 Мультиплатформенность
- ✅ Android приложение (APK)
- ✅ Web-версия (Firebase Hosting)
- ⏳ iOS (в разработке)

### 🌍 Мультиязычность
- 🇮🇱 Иврит (עברית) - основной язык
- 🇷🇺 Русский
- 🇬🇧 English

### 👥 Система ролей
Приложение поддерживает 5 типов пользователей:
1. **Super Admin** - полный контроль системы
2. **Admin** - управление пользователями и настройками
3. **Dispatcher** (Диспетчер) - создание и управление маршрутами
4. **Driver** (Водитель) - выполнение доставок
5. **Warehouse Keeper** (Кладовщик) - управление складом

## Основные преимущества

1. **Автоматизация** - автоматическое распределение заказов по водителям
2. **Оптимизация** - расчет оптимальных маршрутов
3. **Отслеживание** - GPS-трекинг водителей в реальном времени
4. **Удобство** - интуитивный интерфейс на нескольких языках
5. **Безопасность** - Firebase Authentication и App Check
6. **Масштабируемость** - поддержка нескольких компаний
